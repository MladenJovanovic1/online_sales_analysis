# Online Sales Analysis

## Opis projekta

Ovaj projekat je razvijen u Pythonu koristeći objektno-orijentisano programiranje (OOP) i Git za upravljanje verzijama.

## Klase

### Product
Predstavlja proizvod sa nazivom, cenom i količinom.

### ProductManager
Upravlja listom proizvoda.
- Dodavanje proizvoda
- Prikaz proizvoda
- Uklanjanje proizvoda
- Izračunavanje ukupne vrednosti inventara

### Cart
Upravlja korpom kupca.
- Dodavanje proizvoda u korpu
- Prikaz sadržaja korpe
- Izračunavanje ukupne vrednosti korpe

## Funkcionalnosti

- Upravljanje proizvodima
- Upravljanje inventarom
- Upravljanje korpom kupca
- Izračunavanje ukupne vrednosti proizvoda
- Izračunavanje ukupnog iznosa za naplatu

## Tehnologije

- Python
- Git
